# Agentic AI App

This project uses CrewAI to build an agentic AI applicatio  n.

## Setup

1. Create a virtual environment:
    ```bash
    python3 -m venv vent
    source vent/bin/activate
    ```

2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3. Run the application:
    ```bash
    python -m src.main
    ```
